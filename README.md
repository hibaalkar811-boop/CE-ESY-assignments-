# Assignment 1 - Circular Buffer

## Description
This project implements a Circular Buffer in C.

## Features
- Initialization
- Write (enqueue)
- Read (dequeue)
- Handle overflow and underflow

## How it works
- User enters name
- "CE-ESY" is added
- Data stored in buffer
- Then read and printed
